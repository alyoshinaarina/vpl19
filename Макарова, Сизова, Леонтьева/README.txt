Добрый вечер! Перед вам программа, которая транскрибирует слова.
Принцип действия:
Программа различает гласные и согласные, гласные делит на ударные/безударные, согласные – на мягкие/твёрдые.
Цикл действует пока не пройдёт по всем буквам.
Заменяет букву на соответствующий ей звук.
Выводится транскрипция слова. 
Примеры рабочих слов: кошка. сыро, каналы, вода, тело, мясо
