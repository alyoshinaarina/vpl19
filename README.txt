Добрый день! Перед Вами программа, позволяющая найти в заданном тексте коллокаты для введенного пользователем слова.  
Принцип действия программы:
1) Программа принимает на вход слово и путь к текстовому файлу 
2) Происходит обработка текста и преобразование его в список слов
3) Затем для полученного слова составляется словарь коллокатов: слов, употребляющихся справа/слева от данного слова в тексте. Алгоритм реализован через циклы:сначала программа ищет слово в тексте, если его нет - появляется сообщение об ошибке, если слово есть, то с помощью циклов, списков и словарей происходит создание словаря коллокатов, его ранжирование, вывод на экран пользователю
4) Результаты сохраняются в csv файл
