Добрый день! Перед Вами программа, позволяющая найти в заданном тексте коллокаты для введенного пользователем слова.  
Принцип действия программы:
1) Программа открывает для чтения текстовый файл
2) Текст очищается от посторонних символов, все буквы переводятся в нижний регистр, из полученного текста формируется список, состоящий из его слов
3) Программа просит пользователся ввести слова, у которого будут искаться коллокаты
4) Далее, в случае если введеное пользователем слово есть в тексте, программа продолжает работу, если нет - прекращает функционировать и выводит пользователю сообщение: 'В тексте нет заданного слова...'
5) Если введенное пользователем слово нашлось в тексте, программа начинает цикл по всем элементам списка слов, сравнивает текущее слово с искомым: если они тождественны, то программа заносит в список предыдущее и последующее для данного слова, а также удаляет из списка само слово, чтобы можно было работать дальше с функией определения элемента по индексу
6) Формируется словарь с ключами, являющимися словами-коллокатами, и значениями - их частотой в тексте. Теперь этот словарь нужно представить в ранжированном виде  
7) Формируется пять пустых вспомогательных списков: список значений, список индексов, ранжированный список значений, список ключей и ранжированный список ключей 
8) Все значения полученного в пункте 6 словаря заносятся в список значений
9) Из них определяется максимальное
10) Затем программа выполняет итерации цикла до тех пор, пока переменная-счетчик максимум не станет равна единице. Цикл заключается в поочередном отборе значений списка значений в ранжиованный список значений по критерию их предполагаемого равенства текущему значению переменной максимум и в поочередном отборе значений индексов значений списка значений, в случае если текущее значение переменной значения равно текущей переменной максимум.
11) Каждый ключ полученного в пункте 6 словаря заносится в список ключей. 
12) Ранжированный список ключей формируется из списка ключей в порядке, который задает список индексов.  
13) Наконец формируется итоговый словарь, ключами которого являются элементы  ранжированного списка ключей и ранжированного списка значений.
14) С помощью цикла на экран пользователю выводится полученный ранжированный список.
